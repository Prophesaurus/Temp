Hilfe

Hauptseite


Auf der Hauptseite werden von allen Sensoren ihre Namen und die letzten 10 Temperaturen angezeigt.

Bei einem Klick auf eine der Sensorboxen wird auf eine Detailansicht des jeweiligen Sensors weitergeleitet.

In der oberen rechten Ecke steht ein Knopf zum Ausloggen zur Verfügung.

Sensorseite

Jede Sensorseite ist auf die gleiche Art und Weise aufgebaut.

Es werden die letzten 10 Temperaturen angezeigt, sortiert von alt nach neu.

Die Temperaturen werden farblich rot markiert, wenn sie über der angegeben Maximaltemperatur liegen.

Darunter stehen verschiedene Statistiken für den Sensor.

Die Maximaltemperatur kann vom Nutzer angepasst werden. Das Ändern wird in einer separaten log-Datenbank gespeichert und kann von Administratoren ausgelesen werden.

Der Aktualiseren-Knopf lädt die Daten aus der Datenbank neu.
